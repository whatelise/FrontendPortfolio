Projects

Heyven

Heyven is a platform built by a team of 4, focused on fostering neighbor to neighbor relationships and communication. It was built using React, Sass, and Firebase. Heyven features the abilities to group chat, direct message, and create and filter events.

Foobar

Foobar is two part solution, a dashboard and an order form, created for a bar. The dashboard is created for both the manager and the employees to understand the current status of the bar at any time. The cohesive order form system allows customers to order and edit their orders comfortably. This project was built using React and Sass.

TAH DA

Tah Da is a configurator in which the user can interactively choose the colors and patches they want to decorate their hat with.

Hacked Hogwarts

Hacked Hogwarts is a detailed database in which the user can filter and sort a list of students, search from the list, and see the count of the current list.

Funkel

Funkel is an online candle webshop in wich the product list is connected to restdb.io and the user is able to filter the product list by collection.

Ideel Kaffe

Ideel Kaffe is a startup company based in Copenhagen that focuses on creating recyclable coffee pods. This website was created as a redesign for Ideel Kaffe.

Pizzarazzi

Pizzarazzi is an interactive computer game that utilizes CSS animations and Javascript.

Silfen

Silfen is a fashion company located in Copenhagen. As a class assignement, my group and I did a redesign of Silfen's website and incorporated their new sub-brand Silfen Play.

About Me

My name is Elise and I am studying Multimedia Design at KEA.

I was born and raised in Houston, Texas, and then moved to Florence, Italy for 6 years before moving to Copenhagen in 2020. I have a bachelors degree in photography from St. Edwards University in Austin. After completing two years in the Multimedia Design program at KEA, I have learned HTML, CSS, javascript, React, design principles, UX research techniques, and more.

Likes: spicy food, sea animals, sad songs, reality tv.
Dislikes: pruney fingers, animal tourism, anchovies.
